### 概述

  OpenAI API 几乎可以应用于任何涉及理解或生成自然语言或代码的任务。我们提供一系列具有不同功率水平的`models`，适用于不同的任务，并能够微调您自己的定制模型。这些模型可用于从内容生成到语义搜索和分类的所有内容。
